# DariusBartenderAPI
API de Mixologia
Este es el primer lanzamiento de nuestra API.
La API proporciona información sobre cristalería, métodos de preparación y algunas bebidas alcohólicas. Con esta API, los desarrolladores podrán acceder a una amplia variedad de datos relacionados con la coctelería y el trabajo de un bartender.

Características principales:

Información detallada sobre diferentes tipos de cristalería utilizada en coctelería.
Métodos de preparación y técnicas utilizadas para crear bebidas.
Datos sobre bebidas alcohólicas.
En este primer lanzamiento, queremos informar que por el momento solo están habilitadas las peticiones GET. Esto significa que los desarrolladores pueden realizar consultas y obtener datos de manera segura, pero no podrán realizar modificaciones o añadir nuevos registros.

Este primer lanzamiento marca el comienzo de nuestro proyecto y esperamos que la API sea útil para los entusiastas de la coctelería, barmans profesionales y desarrolladores que deseen incorporar información de calidad sobre bebidas y cristalería en sus aplicaciones.

¡Explora la API, experimenta con los datos y disfruta de la diversidad de la coctelería!
